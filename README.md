# EMC_MINER

electron forge + vite

## Getting Started

```bash

#====== For DEVELOP ============
# step1: install dependencies
yarn
# step2: start electron process
yarn start

#====== For PUBLISH ============
# step1: build vue-project 
yarn package

```

## package
```bash
# normal package
yarn make
# x64 package
yarn make:x64
```
